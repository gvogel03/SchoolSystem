School System Readme.txt

1. Install PostgreSQL and pgAdmin4
2. Initialize a new server and database named "school" and password 1234
3. Copy and paste the contents of create.sql into pgAdmin4's query tool
4. Copy and paste the contents of populate.sql into pgAdmin4's query tool
5. Open up the Java files in your preferred IDE
6. Add postgresql-42.5.1.jar to your project path
7. Make sure line 133 of login.java routes to the corrrect database path (jdbc:postgresql://localhost:5432/School)
8. Compile and run java files
9. Log in
